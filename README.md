# 🏠 KIIMU OS

### A premium Apple-inspired dashboard for Home Assistant

Modern • Minimal • Glassmorphism • Responsive

## ✨ About

KIIMU OS is a complete frontend experience for Home Assistant.

Inspired by Apple Home, visionOS and modern minimal UI.

## Features

- Room-based navigation
- Bubble Card + Mushroom
- Browser Mod popups
- Responsive layout
- Energy & Media dashboards
- Homey integration
